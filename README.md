# Bookstore Management System (DSA Coursework)

## Overview

This project is a console-based Bookstore Management System developed for a Data Structures and Algorithms (DSA) coursework. It demonstrates the application of fundamental data structures and algorithms in managing a book catalog and processing customer orders.

The system utilizes Linked List, Queue, and Stack, along with algorithms such as Binary Search and Merge Sort to perform searching, sorting, and order handling efficiently.

## Key Features

- Manage book catalog  
- Process orders using Queue (FIFO)  
- Support undo operations using Stack (LIFO)  
- Search books by title using Binary Search  
- Sort books using Merge Sort  

## Purpose

The project aims to bridge theoretical DSA concepts with practical implementation, focusing on correctness, efficiency, and algorithmic trade-offs.
